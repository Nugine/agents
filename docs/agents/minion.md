# Minion

ReAct 范式通用终端助手。

| 字段     | 值                          |
| -------- | --------------------------- |
| Kind     | `minion`                    |
| 标题     | `⬦ Minion`                  |
| 范式     | ReAct（推理 → 行动 → 观察） |
| 默认模型 | medium 级                   |

## 定位

Minion
是平台的主要入口智能体。它拥有全部平台工具，适合日常终端操作、文件管理、后台任务、子智能体调度等场景。系统提示词中包含模型选择指南，可根据任务复杂度自动选择合适的子智能体模型。
